## Machine Learning

*Python + scikit-learn appear in 71% and 34% of ML job postings respectively — [Source](https://365datascience.com/career-advice/career-guides/machine-learning-engineer-job-outlook-2025/)*

| Skill | Demonstrated By | Recruiter Relevance |
|-------|-----------------|---------------------|
| Supervised learning (classification) | Logistic Regression on labeled sentiment data | Core ML competency |
| Train/test splitting | 80/20 split with proper data partitioning | Prevents overfitting — fundamental practice |
| Model evaluation metrics | Accuracy, precision, recall, F1-score, confusion matrix | Expected in all ML interviews |
| Feature engineering | TF-IDF with max_features tuning | Key differentiator for model performance |
| Model serialization | Saving models with joblib for deployment | Production-readiness skill |
| Data leakage prevention | Fit on train, transform on test | Shows understanding of ML best practices |
